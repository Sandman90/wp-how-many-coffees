In the future there will be different styles that canbe configured with a simple click.= Can I quantify only the number of coffees? =Basically, the plugin provides different types of different icons,including spoons, coffee cups, sugar cubes, but you can also inserticons from external libraries, inserting them in the configuration.== Translations ==* English== Changelog === 1.0.0 - 2023.04.16 =* Initial release.
